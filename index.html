Capabilities

Three formats, one panel

Everything your documentation needs, without leaving the editor.

01
Write

Open any .md file and get a live side-by-side preview. Mermaid blocks inside markdown are rendered inline with pan & zoom. Six diagram themes including auto dark/light.

02
Draw

.excalidraw files open as a full canvas editor — integrated with VS Code's undo, redo, Ctrl+S, and rename. Saved as human-readable JSON that works seamlessly with version control.

03
Organize

Auto mode discovers docs across your entire workspace. Folder mode keeps everything under .docspace/ with automatic scaffolding. Create, rename, and delete files from the sidebar.
